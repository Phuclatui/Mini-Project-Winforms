## Features

Drag and drop the ball using the mouse.

Automatic falling animation using Timer.

Win detection when the ball reaches the target area.

Lose detection when the ball leaves the game area.

Reset game functionality.

Exit confirmation dialog.

Demonstrates WinForms mouse events and timer-based animation.
